<h1 align=center>🌐 Mais um Projeto Responsivo</h1>

  <li>HTML5</li>
  <li>CSS (com Media Queries para responsividade)</li>
</ul>

<ul>

</li>
<li>Estilização com CSS</li>

</ul>
